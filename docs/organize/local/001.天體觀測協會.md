# 天體觀測協會

## 簡述
總部設於「陣列岩」的學術組織，由[[004.埃拉托|埃拉托家族]]主導，致力於星象觀測、曆法修訂與高階航海術的研究。協會成員相信天體的運行規律與大地的幾何結構存在著神聖的對應關係，透過觀測星辰，他們能預測極端潮汐、洋流變動乃至於未來的災厄。

協會與[[003.諾摩斯|諾摩斯家族]]維持著緊密的合作關係，為城市提供精確的時間基準與導航數據。其核心設施「星軌觀測塔」擁有全大陸最精密的透鏡組，吸引了無數追求真理的學者與航海家前來交流。

### 其他備註
協會的徽章是一個嵌入六分儀的等邊三角形，象徵著天空、海洋與數學的結合。
